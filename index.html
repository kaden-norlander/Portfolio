The Rift Ticketing App

“The Rift” is a ticket app for events created by organizations. All users are able to purchase tickets for events and receive a QR code by email upon completion of their tickets. This code can be used for admission by a simple scan of the code. Transactions are made with STRIPE.

Earth Motion Tracker App

"Earth Motion" is a React-based application that utilizes the USGS (U.S. Geological Survey) API to monitor seismic activity across the globe. It provides users with a visual and data-driven summary of earthquake events, plotting them on a high-contrast map for geographical context.
