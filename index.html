Projects

The Rift Ticketing App

The Rift is a high-performance ticketing ecosystem programmed with Django. It bridges the gap between event organizers and attendees by automating the entire purchase loop. Integrated Stripe API for secure payment processing and developed a custom backend logic that generates unique, encrypted QR codes sent via automated email. Organizers can verify admission in real-time with a simple scan, reducing wait times and preventing ticket fraud.

Earth Motion Tracker App

Earth Motion is a dynamic React application designed to monitor global seismic activity in real-time. By leveraging the USGS API, the app synchronizes live earthquake data and visualizes it on a high-contrast, interactive map. We focused heavily on UI/UX design, implementing custom filtering logic that allows users to sort events by magnitude and proximity. The project showcases my ability to handle asynchronous data streams and render complex geographical datasets efficiently.
